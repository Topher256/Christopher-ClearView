    <div class="card-body" style="line-height:1.8;color:var(--text-muted);">
      <p style="margin-bottom:1rem;">
        <strong style="color:var(--text);">VisionSum</strong> is an undergraduate capstone prototype that demonstrates how 
        computer-vision–based temporal analysis can dramatically reduce the burden of 
        reviewing CCTV footage. Traditional surveillance review requires watching every 
        second of recorded video — a time-consuming and attention-fatiguing task. 
        VisionSum automates this by detecting periods of genuine activity and 
        discarding idle background footage.
      </p>
      <p style="margin-bottom:1rem;">
        The system processes uploaded video files through a four-stage pipeline: 
        <em>preprocessing → temporal analysis → event detection → summarization.</em>
        The output is a shorter video that chronologically preserves all meaningful 
        events at the user-chosen compression level.
      </p>
      <p>
        All processing runs locally on the server — no footage is sent to external 
        services. Users authenticate before accessing any feature, and an admin panel 
        provides oversight of all uploaded assets.
      </p>
    </div>
  </div>
